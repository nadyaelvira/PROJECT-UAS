# Plan: Halaman Login / Sign-In dengan Forgot Password

## Ringkasan
Membuat halaman Login (`/login`) dan Register (`/register`) terpisah dengan desain sesuai tema landing page (navy blue gradient). Frontend-only вҖ” tanpa integrasi backend. Termasuk opsi "Forgot Password" di halaman login.

---

## File yang Dibuat/Diubah

### Baru:
| File | Fungsi |
|------|--------|
| `app/(auth)/layout.tsx` | Layout bersama untuk halaman auth (gradient background, no navbar/footer) |
| `app/(auth)/login/page.tsx` | Halaman login |
| `app/(auth)/register/page.tsx` | Halaman register/daftar |

### Diubah:
| File | Perubahan |
|------|-----------|
| `components/LandingNavbar.tsx` | Tombol Login вҶ’ `<Link href="/login">`, Sign In вҶ’ `<Link href="/register">` |
| `context/SapaLanguageContext.tsx` | Tambah key translasi untuk form login/register |

---

- Link "Lupa password?" di halaman login

Ubah tombol `<button>` menjadi `<Link>`:

```tsx

// Login button
<Link href="/login" className="px-4 py-2 text-sm font-medium text-white hover:text-white/80 transition-colors">
  {t('nav.login')}
</Link>

// Sign In button
<Link href="/register" className="px-4 py-2 text-sm font-medium text-white bg-[#1e3a8a] rounded-lg hover:bg-[#172554] transition-colors shadow-md">
  {t('nav.signIn')}
```

## Verifikasi

1. Jalankan `pnpm dev` / `next dev`
2. Buka `http://localhost:3000/login` вҖ” cek tampilan, input, toggle password, link lupa password
3. Klik "Lupa password?" вҖ” muncul form reset, isi email, submit вҶ’ pesan sukses
4. Klik "Daftar" link вҶ’ navigasi ke `/register`
5. Buka `http://localhost:3000/register` вҖ” cek form, link "Masuk" вҶ’ navigasi ke `/login`
6. Klik tombol Login/Sign In di navbar вҶ’ navigasi ke halaman yang benar
7. Cek toggle bahasa ID/EN berfungsi
8. Cek responsivitas mobile
