## Perubahan Detail

### 1. Hero Section (`app/sapa/page.tsx` - baris 14-90)

**Layout saat ini:** Grid 2 kolom, kiri = teks + tombol, kanan = dashboard mockup SVG

**Layout baru:** Grid 2 kolom (lg:grid-cols-2)
- **Kiri:** Kosong, menjadi area untuk background image `lansia.jpg` (diletakkan sebagai gambar latar atau elemen dekoratif)
- **Kanan:** Konten teks:
  1. **"SAPA"** — teks besar, font extra bold, warna biru (primary-600)
  2. **"Solusi Aman untuk Pemantauan Lansia"** — subtitle di bawah SAPA
  3. **"Lindungi Orang Tersayang dengan Pemantauan Lokasi Secara Real-Time"** — headline utama
  4. **Deskripsi:** "SAPA membantu keluarga memantau lokasi lansia secara langsung, mengatur zona aman, dan menerima notifikasi ketika lansia keluar dari area yang telah ditentukan."
  5. **CTA Button:** "Pelajari Lebih Lanjut →" dengan ikon panah, mengarah ke section `#about` (Apa itu SAPA?)

**Background:** Gradasi biru-putih (`bg-gradient-to-br from-primary-50 via-white to-primary-100`)

### 2. LandingNavbar Logo (`components/LandingNavbar.tsx` - baris 31-39)

**Saat ini:** SVG icon biru + teks "SAPA"
**Baru:** `<Image src="/logo.png" alt="SAPA Logo" width={40} height={40}` + teks "SAPA"

### 3. Final CTA Section (`app/sapa/page.tsx` - baris 395-413)

**Saat ini:** Background gradient biru solid
**Baru:** Background image `keluarga.png` dengan overlay gelap, teks:
- "Mulai Lindungi Orang Tersayang Bersama SAPA"
- Tombol "Daftar Sekarang"

### 4. Translasi (`context/SapaLanguageContext.tsx`)

Tidak perlu tambah key baru — semua teks yang diminta sudah ada di translations:
- `hero.subtitle` = "Solusi Aman untuk Pemantauan Lansia"
- `hero.title` = "Lindungi Orang Tersayang dengan Pemantauan Lokasi Secara Real-Time"
- `hero.description` = teks deskripsi SAPA
- `cta.title` = "Mulai Lindungi Orang Tersayang Bersama SAPA"
- `cta.register` = "Daftar Sekarang"

Tambahan key baru:
- `hero.ctaLearnMore` = "Pelajari Lebih Lanjut" / "Learn More" (untuk tombol CTA hero)
