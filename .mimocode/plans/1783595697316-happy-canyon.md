# Plan: Landing Page SAPA - Solusi Aman untuk Pemantauan Lansia

## Overview
Membuat landing page modern untuk aplikasi SAPA (Solusi Aman untuk Pemantauan Lansia) dengan gaya desain profesional, bersih, dan bernuansa kesehatan.

## Keputusan Desain
1. **Lokasi**: Route baru di `/sapa` (bukan menggantikan homepage)
2. **Bahasa**: Bilingual Indonesia & English dengan toggle bahasa, Indonesia sebagai default
3. **Ilustrasi**: SVG inline untuk ikon, gambar dari `/public` untuk ilustrasi kompleks (akan disesuaikan nanti)

## Tech Stack
- Next.js 16 + React 19
- Tailwind CSS 4 (sudah terkonfigurasi)
- Font: Geist (sudah terkonfigurasi)
- Warna: Biru (#2563EB) + Putih (sudah ada di tailwind config)

## File yang akan dibuat

### 1. `app/sapa/page.tsx` (UTAMA - Landing Page)
Halaman landing page lengkap dengan semua section:

#### Structure:
```
1. Navbar
   - Logo SAPA (SVG custom)
   - Menu: Beranda, Fitur, Cara Kerja, Tentang, Kontak
   - Toggle bahasa (ID/EN)
   - Tombol: Login + Sign In/Daftar

2. Hero Section
   - Subtitle: "Solusi Aman untuk Pemantauan Lansia"
   - Headline: "Lindungi Orang Tersayang dengan Pemantauan Lokasi Secara Real-Time"
   - Deskripsi tentang fitur utama
   - Tombol: "Daftar Sekarang"
   - Ilustrasi: Mockup dashboard di sisi kanan

3. Section "Mengapa Menggunakan SAPA?"
   - Penjelasan tentang demensia dan risiko tersesat
   - Ilustrasi lansia

4. Section "Fitur Unggulan"
   - 3 kartu: Monitoring Lokasi Real-Time, Zona Aman (Geofencing), Notifikasi Otomatis
   - Ikon modern + deskripsi

5. Section "Cara Kerja"
   - 5 langkah dengan ilustrasi
   - 1. Buat akun & login
   - 2. Hubungkan perangkat lansia (pairing)
   - 3. Tentukan zona aman
   - 4. Pantau lokasi real-time
   - 5. Notifikasi otomatis

6. Section "Keunggulan SAPA"
   - Poin-poin keunggulan dalam grid
   - Mudah digunakan, antarmuka ramah lansia, monitoring akurat, dll

7. Call to Action
   - Headline persuasif
   - Tombol: "Daftar Sekarang" + "Masuk"

8. Footer
   - Logo SAPA
   - Menu navigasi
   - Info kontak & email
   - Media sosial
   - Copyright
```

## Design System

### Warna
- Primary: `#2563EB` (Biru) - untuk elemen interaktif
- Background: `#FFFFFF` (Putih)
- Text: `#111827` (Hitam Gelap)
- Text Secondary: `#6B7280` (Abu-abu)
- Accent: `#22C55E` (Hijau - untuk status aman)

### Typography
- Headline: Bold, 3rem
- Subheading: Semi-bold, 1.5rem
- Body: Regular, 1rem

### Spacing & Layout
- Section padding: py-20 (5rem)
- Max width: max-w-7xl (80rem)
- Grid gap: gap-8

### Visual Style
- Rounded corners: rounded-2xl (1rem)
- Soft shadows: shadow-md, shadow-lg
- Gradient backgrounds untuk CTA section
- Banyak whitespace
- Ilustrasi konsisten menggunakan SVG

## Implementasi

### Step 1: Buat SapaLanguageContext.tsx
```tsx
- Context untuk toggle bahasa ID/EN
- Default: Indonesia
- Translasi untuk semua teks landing page
```

### Step 2: Buat LandingNavbar.tsx
```tsx
- Logo SAPA dengan SVG icon
- Menu dengan smooth scroll anchor
- Toggle bahasa ID/EN
- Responsive: mobile hamburger menu
- Tombol Login & Sign In di pojok kanan
```

### Step 3: Buat LandingFooter.tsx
```tsx
- Logo & deskripsi singkat
- Menu navigasi (kolom)
- Info kontak (kolom)
- Social media icons
- Copyright
```

### Step 4: Buat app/sapa/layout.tsx
```tsx

- Import SapaLanguageContext
```

### Step 5: Buat app/sapa/page.tsx
```tsx
- Import LandingNavbar & LandingFooter
- Hero section dengan ilustrasi SVG
- Section-section dengan grid layout

```

## Ilustrasi
Menggunakan SVG inline untuk:
- Ikon fitur (lokasi, GPS, notifikasi)
- Ikon langkah-langkah
- Ikon keunggulan
- Logo SAPA

Untuk ilustrasi kompleks (mockup dashboard, gambar lansia):
- Akan menggunakan placeholder dari `/public`
- User akan menyediakan gambar nanti

## Responsive Design
- Mobile: Stack layout, hamburger menu
- Tablet: 2-column grid
- Desktop: Full layout dengan sidebar
